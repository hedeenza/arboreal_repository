Magikarp Ground Mission (picoCTF 32)

Challenge Author: SYREAL

Category: General Skills

Difficulty: Easy

Description

Do you know how to move between directories and read files in the shell?

Process / Notes

  1. Start the challenge instance
  2. ssh into the container
  3. ls to open the session
  4. There’s a text file with the beginning of the flag + directions to part 2 of 3
  5. ls -la to check for anything hidden –> nothing there
  6. cat the first part –> we have the first part of the flag
  7. cat the instructions –> tells us to go to the root directory ‘/’
  8. In the root, we have part 2 of 3 of the instructions and the second part of the flag
  9. cat the instructions –> tells us to go to the home directory ‘~’
  10. cd ~
  11. We have the last part of the flag
  12. The flag is assembled!

8 minutes 17 seconds to complete

Hints

Core Lessons

  1. Understand how to view directory contents
  2. Understand how to move around the file system
  3. Understand how to read out a file’s contents
