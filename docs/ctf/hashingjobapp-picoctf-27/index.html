HashingJobApp (picoCTF 27)

Challenge Author: LT ‘SYREAL’ JONES

Category: General Skills

Difficulty: Easy

Description

If you want to hash with the best, beat this test!

Process / Notes

  1. nc into the challenge instance
  2. md5 has the provided text ‘cold pizza’   $echo 'cold pizza' | md5sum
  3. Why didn’t it work?
  4. Tried again
  5. Looked up the md5sum command and found that you may need to include the ‘-n’ option on the echo to avoid the encoding of the new line character at the end   $echo -n 'cold pizza' | md5sum
  6. That worked!
  7. Two more quick md5sums and the flag is provided

8 minutes 3 seconds to complete

Hints

  1. You can use a commandline tool or a web app to has text
  2. Press CTRL and c on your keyboard to close your connection and return to the command prompt

Core Lessons

  1. Understand what a hash is
  2. Understand how to redirect desired input to the hashing protocol to get your desired output
