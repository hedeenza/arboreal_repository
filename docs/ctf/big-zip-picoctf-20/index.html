Big Zip (picoCTF 20)

Challenge Author: LT ‘SYREAL’ JONES

Category: General Skills

Difficulty: Easy

Description

Unzip this archive and find the flag

Process / Notes

  1. wget and unzip
  2. TONS of folders and sub-folders - a big zip indeed
  3. Looks like a recursive grep is the solution!
  4. grep -r 'pico'
  5. Returned a long, very buried file, a brief message, and the flag

3 minutes 27 seconds to complete

Hints

  1. Can grep be instructed to look at every file in the directory and its subdirectories?

Core Lessons

  1. Understand how to use grep’s recursive option to find your target in a large number of files
