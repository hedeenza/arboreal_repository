Tab, Tab, Attack (picoCTF 34)

Challenge Author: SYREAL

Category: General Skills

Difficulty: Easy

Description

Using tab-complete in the Terminal will add years to your life, esp. when dealing with long rambling directory structures and filenames: Addadshashanamu.zip

Process / Notes

  1. wget the zip
  2. unzip - definitely long file names
  3. tree - only one file path but with long file names
  4. Going to use find -name on the final file name then cd to the last directory to save even more time!
  5. It’s not a text file, it’s a 64-bit pie executable
  6. strings showed some things but not what we’re looking for
  7. hd and exiftool don’t seem to be the way either.
  8. Going to open it up in the web terminal because I’m always suspicious of the file type no matter the origin
  9. Running the file produced the flag!

7 minutes 18 seconds to complete

Hints

Core Lessons

  1. Understand how to use tab-complete to save on some typing when the coming content is unambiguous
  2. Understand how to interact with the final file
